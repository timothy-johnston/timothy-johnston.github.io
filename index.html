TIMOTHY JOHNSTON

Software Developer from Pittsburgh, Pennsylvania

"I could either watch it happen or be a part of it."

- Elon Musk

I choose to be a part of it.

Builder. Learner. Yinzer.

Full-stack developer. Full-time innovator.

I love problem solving, hypothesizing, breaking things, and fixing things. I'm a member of Pittsburgh's thriving tech community, and I couldn't be more proud to play a part in this fantastic city's tech rennasaince.

My strongest interests lie in API development and data analysis + visualization, and I'm a major proponent of agile development and thoroughly tested code.

When I'm not programming, you'll find me running around a soccer field, setting up camp in the woods, or cooking up something real spicy in the kitchen.

My Strongest Skills & Favorite Technologies

Portfolio

ThirstyCat: Hardware

Raspberry Pi, Python
twitter.com/TheThirstyCat

Pressure-activated camera & Twitter bot that detects when my cat gets a drink, captures a photo, and automatically tweets the photo using the Twitter API.

ThirstyCat: Software

Java & Spring Boot

Full-stack web app allowing users to view ThirstyCat photos and statistics. Users can create an account to receive automated email updates, favorite pictures, and submit photos of their own.

Personal Portfolio Page

HTML, CSS, Javascript

Static website built to serve as my online portfolio and blog. Features modern styling, a contact form, and a responsive layout.

Virtual Interview

Java

Command line application that simulates a mock interview by asking users a series of behavioral interview questions. Users may add, modify, or delete questions to tailor the interview practice to their needs.

WeatherViz

Java, Spring MVC

Collaboratively built forecast app developed with the agile methodology. Users may view visualizations of current and historic weather conditions across the US or create an account to receive personalized forecasts.

Nat'l Parks Reservation

Java, PostgreSQL

Simulated reservation system that allows users so search, read about, and reserve campsites at a variety of US National Parks.
